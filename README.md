# Моя реализация редиса на Go.

## Возможности приложения

- Привязка к TCP-порту и прослушивание входящих соединений
- Ответы на базовые команды, такие как `PING` и `ECHO`
- Разбор Redis Protocol (RESP) из клиентских запросов
- Одновременная работа с несколькими клиентами
- Реализация команд `SET` и `GET` для сохранения и получения данных
- Репликация Redis
- Персистентность данных в формате RDB
- Атомарные транзакции
 
С этой версией приложения можно взаимодействовать с помощью официального `redis-cli`.
